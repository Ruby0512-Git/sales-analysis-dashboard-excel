# sales-analysis-dashboard-excel
Interactive sales dashboard using Excel, Pivot Tables, Charts and Slicers.
This project is an interactive sales analysis dashboard built using Microsoft Excel.

The dashboard helps analyze:
Total Revenue
Average sales
Order Count
Category wise sales
Region wise sales
Product wise sales
Monthly sales Trends
Salesperson Performance

Tools Used:
Microsoft Excel
Pivot Table
Pivot Charts'
Slicers
KPI Cards

Key Learnings:
Data Cleaning
Data Analysis
Dashboard Building 
Business KPI Reporting 
Interactive Reporting using Slicers.
